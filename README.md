# Fluxora

O **Fluxora** é uma plataforma de gestão financeira pessoal que
automatiza a organização do fluxo de caixa a partir de **comprovantes de
pagamento e registros de renda**, transformando dados dispersos em
**informações claras sobre gastos, desperdícios e oportunidades de
investimento**.

O sistema permite que usuários visualizem sua situação financeira de
forma simples e objetiva, auxiliando no **planejamento financeiro e na
construção de reservas para o futuro**.

------------------------------------------------------------------------

# Visão do Projeto

Muitas pessoas têm dificuldade em controlar seus gastos porque dependem
de **planilhas manuais ou registros incompletos**.

O **Fluxora** resolve esse problema ao:

-   Ler **comprovantes de pagamento**
-   Identificar automaticamente **entradas e saídas**
-   Construir um **fluxo de caixa pessoal**
-   Gerar **análises financeiras e sugestões de investimento**

O objetivo é ajudar o usuário a **entender para onde o dinheiro está
indo** e **como melhorar sua saúde financeira**.

------------------------------------------------------------------------

# Principais Funcionalidades

## Importação de comprovantes

O usuário pode enviar:

-   Fotos
-   PDFs
-   Scans

O sistema extrai automaticamente informações como:

-   valor
-   data
-   estabelecimento
-   tipo de gasto

------------------------------------------------------------------------

## Registro de fontes de renda

O usuário cadastra previamente suas fontes de renda, como:

-   salário
-   prestação de serviços
-   renda extra
-   outros ganhos recorrentes

Essas informações alimentam o cálculo do fluxo de caixa.

------------------------------------------------------------------------

## Fluxo de caixa automático

A partir dos comprovantes e rendas cadastradas, o sistema gera
automaticamente:

-   saldo mensal
-   histórico financeiro
-   balanço entre entradas e saídas

------------------------------------------------------------------------

## Dashboard financeiro

O sistema apresenta visualizações como:

-   principais categorias de gasto
-   gastos recorrentes
-   desperdícios financeiros
-   evolução do saldo ao longo do tempo

------------------------------------------------------------------------

## Recomendações financeiras

Com base nos dados coletados, o Fluxora pode sugerir estratégias como:

-   criação de **reserva de emergência**
-   redução de **gastos desnecessários**
-   direcionamento de recursos para **investimentos**

------------------------------------------------------------------------

# Estudo de Investimentos

O projeto também considera diferentes formas de investimento, como:

-   CDB
-   Poupança
-   Ações
-   Previdência privada

O sistema pode analisar **rentabilidade, risco e perfil financeiro do
usuário** para orientar decisões.
